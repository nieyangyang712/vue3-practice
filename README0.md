# vue3Practice
vue3.0 学习

# 查看版本
$ npm -v
2.3.0

# 升级 npm
cnpm install npm -g

# 升级或安装 cnpm
npm install cnpm -g

# 最新稳定版
$ cnpm install vue@next

# 全局安装 vue-cli
yarn global add @vue/cli
# 或
cnpm install -g @vue/cli

# 安装完后查看版本:
$ vue --version
@vue/cli 4.5.11

# 然后在 Vue 项目中运行：
vue upgrade --next

# 接下来我们创建 runoob-vue3-app 项目
vue create runoob-vue3-app
